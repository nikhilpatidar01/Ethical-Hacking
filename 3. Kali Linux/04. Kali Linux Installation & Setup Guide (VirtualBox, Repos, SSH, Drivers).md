# 🗂️ Kali Linux Repository Setup

### 1. 🆕 Update Package Index

```bash
apt update
```

---

### 2. 🧰 Install Basic Utilities

```bash
apt install apt-transport-https vim curl
```

---

### 3. 🔑 Add Kali Key

```bash
curl -fsSL https://archive.kali.org/archive-key.asc | sudo gpg --dearmor -o /usr/share/keyrings/kali-archive-keyring.gpg
```

---

### 4. 🔧 Edit Source List

```bash
vim /etc/apt/sources.list
```

**Example:**

```bash
deb http://http.kali.org/kali kali-rolling main contrib non-free
```

💡 Full mirror list:
👉 [https://http.kali.org/README.mirrorlist](https://http.kali.org/README.mirrorlist)

---

### 5. 🧹 Clean & Update

```bash
apt-get clean
rm -rf /var/lib/apt/lists/*
apt update
```

---

### 6. 🆙 Upgrade System

```bash
apt upgrade -y
apt full-upgrade -y
```

> 🔁 Reboot if required:

```bash
[ -f /var/run/reboot-required ] && reboot
```

# 📶 Realtek RTL8812AU Wireless Adapter Setup

> Used for Wi-Fi cracking, monitor mode, packet injection

### 1. Update System

```bash
apt update
```

---

### 2. Install Required Dependencies

```bash
apt install dkms git build-essential libelf-dev linux-headers-$(uname -r)
```

---

### 3. Install Driver

```bash
apt install realtek-rtl88xxau-dkms
```

> 💡 DKMS will **auto-compile** on future kernel upgrades

---

### 4. Reboot

```bash
reboot
```
